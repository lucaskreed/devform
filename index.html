Squid's Development Team

Join the team and help create something extraordinary!

  • UI Designer (6%): Designs user interfaces for a smooth player experience.
  • Scripters (5% each): Develop scripts to implement gameplay mechanics.
  • Modelers (4% each): Design and produce 3D models and assets.
  • Audio Producer (4%): Create sound effects and music for the game.
  • Sr. Builders (3% each): Lead environment and map creation.
  • Jr. Builders (1% each): Assist with building tasks.
  • VFX Artists (1% each): Create visual effects for immersive gameplay.
  • Animators (1% each): Develop animations for characters and objects.
  • Studio Manager (1%): Oversee team operations and deadlines.
  • Media Team: Promote the game and manage community engagement (perks only).
  • Sr. Moderators, Moderators, and Trainee Moderators: Paid separately.
  • Fan Map Designer: Collaborate on community-inspired maps (no payment).
  • Studio Director: Leads the team vision and direction (no payment).
  • Artists (1% each): Create concept art and visual assets for the game.

Profits are distributed as follows:

  • UI Designer: 7%
  • Scripters: 5% each
  • Modelers: 4% each
  • Audio Producer: 4%
  • Sr. Builders: 4% each
  • Jr. Builders: 2% each
  • VFX Artists: 1% each
  • Animators: 1% each
  • Studio Manager: 1%
  • Artists: 1% each

General Allocations

  • Group Funds: 40%
  • Advertisements: 20%
  • Member Payouts: 40%
