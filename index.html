Squid's Development Team

Apply now to join the team and contribute to amazing projects!

  • Modelers: Create detailed 3D models and assets for the game.
  • Scripters: Develop scripts to bring game mechanics to life.
  • Builders: Design and build immersive environments.
  • UI Designers: Craft user interfaces for an engaging player experience.
  • Animators: Create smooth animations for character and object movements.
  • Artists: Design concept art and game visuals.
  • Social Media Team: Promote the game and interact with the community.
  • Moderators: Maintain a positive community and enforce rules (paid separately).
  • Financial Manager: Manage finances, track income and expenses, and allocate profits.

Profits are distributed as follows:

  • Scripters: 2/3 (5% each)
  • Modelers: 1/2 (5% each)
  • Artists: 1/1 (4%)
  • UI Designers: 1/2 (4% each)
  • Animators: 0/2 (4% each)
  • Builders: 0/5 (2% each)
  • Social Media Team: 1/4 (1% each)
  • Moderators: Paid separately
  • Financial Manager: 1/1 (4%)
  • Group Funding and Advertising: 40%
